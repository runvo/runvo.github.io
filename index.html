runvo

Run AI coding agents from your phone.

Claude Code, Aider, or any CLI agent — from anywhere via SSH.

GitHub Install Homebrew
Install with Homebrew
brew tap runvo/runvo && brew install runvo
or: curl -fsSL https://raw.githubusercontent.com/runvo/runvo/master/install.sh | bash

How It Works

runvo manages your projects and opens AI agent sessions in tmux. Sessions persist even if you disconnect — pick up right where you left off from any device.

$ runvo

    ▸ RUNVO                         v1.0.0
    ────────────────────────────────

    PROJECTS
   1  ● my-backend    Backend API
   2    my-app        Side project

    - - - - - - - - - - - - - - - -

   +  Add existing
   n  Create new
   -  Remove project
   s  SSH auto-launch  on

  # 1  ← type number, press Enter, start chatting

Setup Guide

Two parts: install runvo on your computer, then connect from your phone. Takes about 10 minutes.

Part 1: Install runvo on your computer

macOS
Linux
Windows (WSL)

1. Install Homebrew (if you don't have it)

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. Install runvo

brew tap runvo/runvo && brew install runvo

3. Install an AI agent

npm install -g @anthropic-ai/claude-code   # or: pip install aider-chat

4. Enable SSH (for phone access)

System Settings → General → Sharing → Remote Login → ON

5. Install Tailscale (for phone access)

brew install --cask tailscale

Open the app, sign in. Note your IP (starts with 100.).

↓ Tailscale for Mac

1. Install dependencies

sudo apt update && sudo apt install -y tmux git curl

For Fedora/RHEL: sudo dnf install tmux git curl

2. Install runvo

curl -fsSL https://raw.githubusercontent.com/runvo/runvo/master/install.sh | bash

Or with Homebrew: brew tap runvo/runvo && brew install runvo

3. Install an AI agent

npm install -g @anthropic-ai/claude-code   # or: pip install aider-chat

4. Enable SSH (for phone access)

sudo apt install openssh-server && sudo systemctl enable --now ssh

5. Install Tailscale (for phone access)

curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up

↓ Tailscale for Linux

1. Install WSL

Open PowerShell as Administrator and run:

wsl --install

Restart your computer. Ubuntu will be installed automatically.

2. Inside WSL, install dependencies

sudo apt update && sudo apt install -y tmux git curl

3. Install runvo

curl -fsSL https://raw.githubusercontent.com/runvo/runvo/master/install.sh | bash

4. Install an AI agent

npm install -g @anthropic-ai/claude-code   # or: pip install aider-chat

5. Enable SSH inside WSL (for phone access)

sudo apt install openssh-server
sudo sed -i 's/#Port 22/Port 2222/' /etc/ssh/sshd_config
sudo service ssh start

Uses port 2222 to avoid conflict with Windows SSH.

6. Install Tailscale on Windows (for phone access)

Install on Windows side. Your WSL shares the Tailscale network.

↓ Tailscale for Windows

Add your first project & test

runvo add my-app ~/Projects/my-app "My App"
runvo

You should see your project listed. Type 1 and press Enter to start chatting.

Part 2: Access from your phone

iPhone
Android

1. Install Tailscale

Sign in with the same account as your computer.

↓ App Store

2. Install Termius (SSH client)

↓ App Store

3. Connect

In Termius: New Host → Hostname: your Tailscale IP → Username & password of your computer.

Find your Tailscale IP on your computer:

tailscale ip -4

Windows WSL users: use port 2222 instead of default.

Start coding from your phone!

Once connected via Termius, just type:

runvo

Pick a project, start chatting with your AI agent. That's it!

Tip: Sessions persist in tmux. If you disconnect (close Termius, lose wifi), just reconnect and run runvo again — your session is still there.

Commands

Daily use

runvo Pick project → chat
runvo 1 Open project #1 directly
runvo 1 c "fix bug" One-shot custom prompt
runvo 1 2 Run preset action #2

Manage projects

runvo new [name] Create new project (mkdir + git init)
runvo add [name path desc] Register existing project
runvo clone <url> [name] Clone repo & register
runvo remove name Remove a project
runvo list List all projects
runvo status Git status dashboard
runvo config Edit config file

Manage prompts

runvo prompts List all prompts
runvo prompt add name Create custom prompt
runvo prompt edit name Edit a prompt
runvo prompt rm name Delete custom prompt

Remote control

runvo send 1 "fix bug" Send prompt to session
runvo peek [n] View output without attaching
runvo attach [name|n] Quick attach to session

Other

runvo setup Setup wizard
runvo sessions Active tmux sessions
runvo kill [name|all] Kill session(s)
runvo history Recent run history
runvo ssh-auto Toggle SSH auto-launch
runvo doctor Check system health
runvo update Update to latest
runvo version Show version
runvo help Full help

Preset Prompts

FAQ

What AI agents are supported?

Any CLI-based AI coding agent. Built-in support for Claude Code and Aider. Others work by setting the prompt flag in ~/.runvo/config.

What platforms are supported?

Server (where runvo runs): macOS, Linux, Windows (via WSL).
Phone (remote access): iPhone and Android — using Tailscale + Termius (both free).

Do I need Tailscale?

Only for phone/remote access. On your computer, runvo works without it. Tailscale is free and creates a secure tunnel — no port forwarding or static IPs needed.

What if I disconnect?

Sessions run in tmux. Close Termius, lose wifi, whatever — your session keeps running. Reconnect and run runvo to pick up where you left off.
