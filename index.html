runvo

Run AI coding agents from your phone.

Claude Code, Aider, or any CLI agent — from anywhere via SSH.

GitHub Install Homebrew
Install with Homebrew
brew tap runvo/runvo && brew install runvo
or: curl -fsSL https://raw.githubusercontent.com/runvo/runvo/master/install.sh | bash

How It Works

runvo manages your projects and opens AI agent sessions in tmux. Sessions persist even if you disconnect — pick up right where you left off from any device.

$ runvo

    ▸ RUNVO                         v1.0.0
    ────────────────────────────────

    PROJECTS
   1  ● my-backend    Backend API
   2    my-app        Side project

    - - - - - - - - - - - - - - - -

   +  Add existing
   n  Create new
   -  Remove project
   s  SSH auto-launch  on

  # 1  ← type number, press Enter, start chatting

Setup Guide

Everything you need to run AI agents from your phone. Takes about 10 minutes.

Part 1: Install on your Mac

Open Terminal on your Mac

Press Cmd + Space, type Terminal, press Enter.

Install Homebrew (if you don't have it)

Paste this and press Enter. Follow the prompts.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Install runvo

brew tap runvo/runvo
brew install runvo

Install an AI agent

Pick one (Claude Code recommended):

npm install -g @anthropic-ai/claude-code

Or Aider: pip install aider-chat

Add your first project

# Existing project:
runvo add my-app ~/Projects/my-app "My App"

# Or create from scratch:
runvo new my-app

new creates the folder, runs git init, and registers it.

Test it works

runvo

You should see your project listed. Type 1 and press Enter to start chatting.

Part 2: Access from your phone

Enable SSH on your Mac

Go to System Settings → General → Sharing → Remote Login and turn it ON.

Install Tailscale

Tailscale creates a private network between your devices. No port forwarding needed.

brew install --cask tailscale

Open the Tailscale app and sign in. Note the IP address shown (starts with 100.).

Install Tailscale on iPhone

Download Tailscale from the App Store. Sign in with the same account.

Install Termius on iPhone

Download Termius from the App Store (free). This is your SSH app.

Connect from iPhone

In Termius: New Host → paste your Tailscale IP → Username: your Mac username → Password: your Mac password.

Find your Tailscale IP:

tailscale ip -4

Run runvo from your phone!

Once connected via Termius, just type:

runvo

Pick a project, start chatting with your AI agent. That's it!

Tip: Sessions persist in tmux. If you disconnect (close Termius, lose wifi), just reconnect and run runvo again — your session is still there.

Commands

Daily use

runvo Pick project → chat
runvo 1 Open project #1 directly
runvo 1 c "fix bug" One-shot custom prompt
runvo 1 2 Run preset action #2

Manage projects

runvo new [name] Create new project (mkdir + git init)
runvo add [name path desc] Register existing project
runvo remove name Remove a project
runvo list List all projects
runvo config Edit config file

Manage prompts

runvo prompts List all prompts
runvo prompt add name Create custom prompt
runvo prompt edit name Edit a prompt
runvo prompt rm name Delete custom prompt

Other

runvo setup Setup wizard
runvo sessions Active tmux sessions
runvo history Recent run history
runvo ssh-auto Toggle SSH auto-launch
runvo update Update to latest
runvo version Show version
runvo help Full help

Preset Prompts

Shipped with runvo — run with runvo 1 <number>:

1 — review Review code, find bugs
2 — fix-lint Fix lint/type errors
3 — test Run test suite
4 — status Git status summary

Create your own: runvo prompt add my-prompt

FAQ

What AI agents are supported?

Any CLI-based AI coding agent. Built-in support for Claude Code and Aider. Others work by setting the prompt flag in ~/.runvo/config.

Do I need Tailscale?

Only if you want phone access. On your Mac, runvo works without it. Tailscale is free and creates a secure tunnel — no port forwarding or static IPs needed.

What if I disconnect?

Sessions run in tmux. Close Termius, lose wifi, whatever — your session keeps running. Reconnect and run runvo to pick up where you left off.

Can I use this on Linux?

Yes. Install tmux and an AI agent, then use the curl installer. Homebrew works on Linux too.
